# 🕰️ Digital Memory Capsule

## 📌 Project Overview

Digital Memory Capsule is a web-based application that allows users to store personal or shared memories (text, images, music, and videos) and unlock them in the future using a time-based system and secure memory keys.

## 🚀 Features

* Create memory capsules with media (image, audio, video links)
* Lock capsules with unlock date
* Unlock using memory key (password)
* Automatic unlock after selected date
* Group capsule for shared memories
* Dashboard to manage all capsules
* Subscription plans (UI simulation)
* Payment interface (simulated)

## 🛠️ Technologies Used

* HTML
* CSS
* Bootstrap
* JavaScript (localStorage, DOM manipulation)

## 📂 Project Structure

* `index.html` → Home page
* `create.html` → Create capsule
* `dashboard.html` → View capsules
* `pricing.html` → Subscription plans
